## 🚀 Configuración Rápida

```
# 1. Crea un repo desde la plantilla: github.com/copilot-dev-days/agent-lab-typescript
# 2. Habilita Pages: Settings > Pages > GitHub Actions
# 3. Elige una opción: clona el repo en VS Code O crea un Codespace desde tu repo
# Luego ejecuta:
# /setup
```

## 💡 Consejos

1. Mantén el navegador abierto para actualizaciones en tiempo real
2. Haz commit del código funcional con frecuencia
3. Usa checkpoints para revertir errores
4. Itera en los planes 2+ veces antes de implementar
