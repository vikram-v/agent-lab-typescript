# 🎮 VS Code GitHub Copilot Agent Lab
[← README](../../README.es.md)

- [ ] VS Code más reciente **v1.113** (sin actualización pendiente)

> **Nivel:** Intermedio
> **Stack:** TypeScript / Vite / Tailwind CSS

Bienvenido a **Bingo Mixer** — un workshop práctico donde transformarás una app de bingo social usando el Agent Mode de VS Code con GitHub Copilot.

---

## 📋 Lista Rápida

- [ ] VS Code más reciente **v1.113** (sin actualización pendiente)

## 🎯 Lo que Aprenderás

| # | Habilidad | Descripción |
|---|-----------|-------------|
| 1 | **Ingeniería de Contexto** | Integra la IA a tu codebase y flujos de trabajo |
| 2 | **Primitivas Agénticas** | Construye un flujo de ingeniería AI-first |
| 3 | **Abundancia Agéntica** | Escala la exploración y acelera el aprendizaje |
| 4 | **Desarrollo Design-First** | Deja que la IA itere en la UI mientras tú guías la visión |

| Parte | Título | Descripción |

---

## 💡 Consejos

1. Mantén el navegador con actualización en tiempo real abierto
2. Haz commit del código funcional con frecuencia
3. Revierte cambios inesperados usando *Checkpoints* y *Undo* del chat, luego intenta de nuevo
