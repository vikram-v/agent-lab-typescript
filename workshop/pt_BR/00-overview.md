# 🎮 VS Code GitHub Copilot Agent Lab
[← README](../../README.pt_BR.md)

- [ ] VS Code mais recente **v1.113** (sem atualização pendente)

> **Nível:** Intermediário
> **Stack:** TypeScript / Vite / Tailwind CSS

Bem-vindo ao **Bingo Mixer** — um workshop prático onde você vai transformar um app de bingo social usando o Agent Mode do VS Code com GitHub Copilot.

---

## 📋 Lista Rápida

- [ ] VS Code mais recente **v1.113** (sem atualização pendente)
  - [ ] Verifique se as atualizações não estão desabilitadas
- [ ] Logado com GitHub (Copilot Free não pode executar cloud agent!)
- [ ] Git & Node instalados
- [ ] Chat aberto e Agent pronto para ação

*Opcional*: Use DevContainer ou WSL.

---

## 🎯 O que Você Vai Aprender

| # | Habilidade | Descrição |
|---|------------|-----------|
| 1 | **Engenharia de Contexto** | Integre a IA ao seu codebase e fluxos de trabalho |
| 2 | **Primitivas Agênticas** | Construa um fluxo de engenharia AI-first |
| 3 | **Abundância Agêntica** | Escale exploração e acelere o aprendizado |
| 4 | **Desenvolvimento Design-First** | Deixe a IA iterar na UI enquanto você guia a visão |

## 📚 Partes do Lab

| Parte | Título | Descrição |
|-------|--------|-----------|
| [**01**](01-setup.md) | Configuração & Engenharia de Contexto | Clone, configure e ensine a IA sobre seu projeto |

---

## 💡 Dicas

1. Mantenha o navegador com atualização em tempo real aberto
2. Faça commit do código funcional com frequência
3. Reverta mudanças inesperadas usando *Checkpoints* e *Undo* do chat, depois tente novamente
