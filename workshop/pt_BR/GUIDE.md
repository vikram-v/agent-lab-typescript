## 🚀 Configuração Rápida

```

# 2. Habilite Pages: Settings > Pages > GitHub Actions
# 3. Escolha uma opção: clone o repo no VS Code OU crie um Codespace a partir do seu repo
# Depois execute:
# /setup
```
